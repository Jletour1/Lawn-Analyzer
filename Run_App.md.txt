# 🚀 How to Run the Lawn Care AI Application

## Quick Start (Choose One Method)

### Method 1: Python Script (Recommended)
```bash
python scripts/start_app.py
```

### Method 2: Batch File (Windows)
```bash
scripts\start_app.bat
```

### Method 3: Direct Command
```bash
npm run dev
```

## What Happens Next

1. **Dependencies Install** (if needed) - Automatic
2. **Server Starts** - Usually takes 10-30 seconds
3. **Browser Opens** - Automatically opens http://localhost:5173
4. **Application Ready** - You can now use the app!

## Using the Application

### 🧑‍💻 User Mode (Default)
- **Upload lawn images** for AI diagnosis
- **Get treatment recommendations** 
- **See similar cases** from the database
- **View confidence scores** and analysis details

### 🔧 Admin Mode
- **Click "Admin" button** in top-right corner
- **Password**: `admin123`
- **Access admin panel** with data management tools

## Troubleshooting

### If you get errors:
1. **Make sure you're in the project folder** (should see package.json)
2. **Install Node.js** if you haven't (version 18+)
3. **Run**: `npm install` to install dependencies
4. **Try again**: `python scripts/start_app.py`

### If browser doesn't open:
- **Manually go to**: http://localhost:5173

### If you want to stop:
- **Press**: `Ctrl+C` in the terminal

## That's It! 🎉
The application should now be running and ready to analyze lawn images!
